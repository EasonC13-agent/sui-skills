# Sui Skills

A collection of AI agent skills for Sui blockchain development.

## Included Skills

- **sui-move** - Sui blockchain and Move smart contract development
- **sui-coverage** - Test coverage analysis with Python tools
- **sui-decompile** - Fetch and analyze on-chain contract source code
- **sui-agent-wallet** - Give your agent its own Sui wallet

## Install individual skills

```bash
npx skills add EasonC13-agent/sui-skills@sui-move
npx skills add EasonC13-agent/sui-skills@sui-coverage
npx skills add EasonC13-agent/sui-skills@sui-decompile
npx skills add EasonC13-agent/sui-skills@sui-agent-wallet
```
